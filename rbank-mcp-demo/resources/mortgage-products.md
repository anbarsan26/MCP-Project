# Mortgage Products Overview

This file summarises the example mortgage products that the demo server can discuss.

## Example products

- annuity-10y-fixed: fixed monthly payments with a 10-year fixed period
- annuity-20y-fixed: fixed monthly payments with a 20-year fixed period
- annuity-30y-fixed: fixed monthly payments with a 30-year fixed period
- variable: rate can change over time

## Training notes

- Use these products to show how an MCP server can provide structured context.
- The prompt workflow can combine product details with rate information and a calculator tool.
