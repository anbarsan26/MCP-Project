# Mortgage Advice Prompt Template

Use this prompt when a user wants help with a mortgage example.

## Workflow

1. Start by using the mortgage advice form tool.
2. Read the mortgage rates resource.
3. Read the mortgage products resource.
4. Use the monthly payment calculator tool.
5. Explain the result in a short, structured way.

## Guidance

- Keep the explanation clear and beginner-friendly.
- Mention that the example is for training and not official financial advice.
- If a detail is missing, make a reasonable assumption and say so.
